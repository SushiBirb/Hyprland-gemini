# Hogwarts-Themed Hyprland Configuration

This repository contains a fully featured, highly customized, and aesthetically cohesive Hyprland desktop environment tailored to a whimsical "Harry Potter" aesthetic.

> **Disclaimer**: This repository and its configurations were generated autonomously by an AI agent as part of an experimental test.

## Features & Architecture

This configuration leverages the powerful **Illogical Impulse** framework (Quickshell, Matugen, Hyprland Lua) to provide a dynamic, highly responsive environment.

- **Dynamic Theming Engine**: Support for 8 distinct color palettes (Gryffindor, Slytherin, Ravenclaw, Hufflepuff - each with Light and Dark modes).
- **Automated Wallpapers**: Cinematic 4K wallpapers generated by AI and injected into Quickshell.
- **Rofi Settings Hub**: A centralized `hogwarts-settings` control panel bound to `SUPER + /` for managing animations, themes, and keybindings.
- **Strict Keybindings**: Adheres exactly to a robust, keyboard-driven workflow (see Keybindings section below).
- **Application Integration**: Transparent `kitty` terminal without cursor trails, and a fully dynamic Zen Browser CSS injection that matches your current house theme.

## Core Software Stack

- **Window Manager**: Hyprland (with Lua plugin)
- **UI Framework**: Quickshell (Illogical Impulse setup)
- **Terminal**: Kitty
- **App Launcher**: Rofi
- **File Manager**: Dolphin
- **Color Engine**: Matugen

## Keybindings

- `SUPER + T`: Launch Terminal (Kitty)
- `SUPER + Q`: Close active window
- `SUPER + [1-9]`: Change workspace
- `SUPER + ALT + [1-9]`: Silently move active window to workspace
- `SUPER + SHIFT + [1-9]`: Move active window to workspace and switch to that workspace
- `SUPER`: Launch App Launcher (Rofi)
- `SUPER + E`: Launch File Explorer (Dolphin)
- `SUPER + /`: Launch the Hogwarts Settings & Keybinds Rofi Menu

## Installation

A fully automated install script is provided to replicate this environment on a fresh Arch system.

```bash
git clone git@github.com:SushiBirb/Hyprland-gemini.git ~/dots-hyprland
cd ~/dots-hyprland
./install.sh
```
